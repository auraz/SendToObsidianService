# Send to Obsidian

Capture selected text to Obsidian inbox via macOS Services or Share Sheet.

## Install

1. Copy script and set your inbox path:
   ```bash
   cp send-to-obsidian.sh ~/bin/
   chmod +x ~/bin/send-to-obsidian.sh
   # Edit INBOX_FILE in ~/bin/send-to-obsidian.sh
   ```

2. Double-click `Send to Obsidian.shortcut` to import

## Use

Select text → right-click → **Services** → **Send to Obsidian**

## Output

```
- 2024-01-15 14:30
  Your selected text
  appears here
```
